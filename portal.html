Setup Guide — How This System Works
🌐 Website — Network Solutions
What you can do on your current site without a developer
  • 1
    Log in to your Network Solutions admin panel at networksolutions.com
  • 2
    Create a new page called "Staff Portal" — set it to password-protected in your page settings
  • 3
    Upload this HTML file as the page content, OR use the HTML/embed block in your page builder and paste the code
  • 4
    Set the page URL to something like safehavenchildrenshome.com/staff
  • 5
    Update the staff portal password in the code (see the STAFF_PASSWORD variable) and share it with employees
⚠️ Network Solutions website builder has limited HTML embed support. If their builder doesn't allow raw HTML pages, the alternative is hosting this file on a free platform like Netlify (netlify.com — free, no code needed, drag-and-drop) and linking to it from your website.
📁 Google Drive + Leave Tracker Sheet
Free, organized, DSWD-ready — and powers the live balance card above
  • 1
    Create "SHCH HR — 201 Files" in Google Drive, shared only with Jackson and the ED. One folder per employee: SANTOS_Maria_2024-01-15. 10 subfolders per employee matching the 201 file tabs.
  • 2
    Create the Leave Balance Sheet. New Google Sheet named "SHCH Leave Balances". Row 1 must have these exact column headers:
    FullName | HireDate | SILEntitlement | SILUsed | SILRemaining | LastUpdated
    One row per employee. FullName must exactly match how employees log in here. Jackson updates weekly from Sprouts PH.
  • 3
    Publish the Sheet as CSV: File → Share → Publish to web → Sheet1 → Comma-separated values (.csv) → Publish → copy the URL
  • 4
    Open this HTML file in Notepad, find YOUR_GOOGLE_SHEET_CSV_URL_HERE and paste the URL. Save and re-upload to your website.
  • 5
    Privacy: The Sheet URL acts as an access key — never post it publicly. Jackson can unpublish it from Google Sheets at any time to immediately disable all balance lookups.
⚠️ RA 10173: This sheet contains employee names and leave data. Do not include salary, government IDs, or health data. Jackson controls access.

⚙️ What You Cannot Do on Network Solutions
Be honest about the platform's limits
  • ✗
    Store employee data — Network Solutions websites don't have databases. Use Google Drive + Google Forms instead.
  • ✗
    Send automatic email reminders — Use Google Forms + Apps Script, or a free tool like Make.com (free tier) connected to Gmail.
  • ✗
    Track leave balances automatically — Use a Google Sheet as a leave tracker. Jackson updates it; the chatbot can explain balances if Jackson keeps it current.
  • ✓
    What it CAN do: host this HTML page, display the chatbot, link to Google Drive, link to Google Forms, display announcements, and serve as a staff-only landing page.
